# User-client



### Getting Started


Clone the repository and install all dependencies

    $ git clone git@github.com:mkoty/user-client.git
    $ cd user-client
    $ npm install

thats it.

### Development

    $ npm start
    

Then visit your [`localhost` on port `3333`](http://localhost:3333)
And try to search by "Sichkar" lastname :)
